# 🎮 L’Ombre du Passé

## 📖 Description

**L’Ombre du Passé** est un jeu narratif d’horreur psychologique à choix multiples développé dans le cadre d’un projet de NSI sur une durée de 2 mois.

Le joueur incarne un personnage dont il choisit le nom, plongé dans un univers sombre où son passé semble le rattraper. Chaque décision influence le déroulement de l’histoire et peut mener à différentes fins.

## 🎯 Objectifs du projet

* Réaliser un jeu interactif avec interface graphique

---

## 🕹️ Gameplay

Le jeu repose sur une narration interactive :

1. Le joueur entre le nom de son personnage
2. Une scène est affichée avec une ambiance horrifique
3. Plusieurs choix sont proposés
4. Le joueur sélectionne une action
5. L’histoire évolue en fonction de ses décisions

👉 Chaque choix peut influencer :

* la progression de l’histoire
* les événements rencontrés
* la fin obtenue

---

## 🌳 Structure du jeu

Le jeu est construit sous forme d’un **arbre de décisions** :

* Chaque scène contient :

  * un texte narratif
  * plusieurs choix possibles
* Chaque choix mène à une nouvelle scène

---

## 🏁 Fins du jeu

Le jeu propose plusieurs fins possibles :

* Bonne fin
* Mauvaise fin
* Fin alternative / secrète

---

## ⚙️ Installation et exécution

### Prérequis

* Python 3.x

### Lancer le jeu

1. Télécharger le projet
2. Ouvrir un terminal dans le dossier
3. Exécuter :

```
python main.py
```
